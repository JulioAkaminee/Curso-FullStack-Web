# Estrutura

Cabeçalho
 Corpo

- No HTML:
  - Head
  - Body

- < !DOCTYPE html >
    - Fala que o HTML vai ser executado na versão mais recente.
- < html lang="pt-br" >  
   - Diz para ler as informações em português.
- < Head > < /Head >
   - Detalhes da aplicação
- < body > < /body >
  
  # Estrutura básica HTML

  < !DOCTYPE html >
  < html lang="pt-br" >
  < head >
   < meta charset="UTF-8">
   < meta name="viewport" content="width=device-width, initial-scale=1.0">
   < title>Document< /title>
  < /head>
  < body>
   
  < /body>
  < /html>
