# Terminal

### Git Bash

- PWD
  - localiza onde estou.
- ls
  - Lista arquivos ou diretórios
- cd
  - Entra em um determinado diretório.
- mkdir
  - Cria um diretório.
- cd..
  - Volta o diretório.
- touch
  - Cria um arquivo.
- rm -r
  - Remove arquivos ou diretórios.
- mv [nome do arquivo] [diretório para onde vai ser movido.]
   - move arquivos.
- mv [nome do arquivo]  [novo nome do arquivo]
   - Renomeia arquivo.
