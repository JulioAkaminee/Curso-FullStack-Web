# Hardware

É a parte fisíca de um equipamento eletrônico.

- Entrada.
  - Teclados, Mouses Etc.
- Saída.
  - Monitor, Fones Etc.
- Armazenamento.
  - Hd,SSD, PenDrive Etc.
- Processamento.
  - Software.

  ---

  # Software
  Um conjunto de instruções que permite controlar um equipamento eletrônico.

  É a **parte lógica** do computador.


- Software de aplicação.
- Software de sistema.
  - Sistema Operacional faz a comunicação entre o hardware e os demais softwares.
  - O sistema operacional cria uma plataforma comum a todos  os programas utilizados.

  ---
  # Numeros Binários

- ON = SIM = 1
- OFF = NÃO = 0
  - A combinação de 1s e 0s em sequências define a informação que está sendo processada ou armazenada.
  - Tabela ASCII dicionário para o código binário. Cada caractere presente na tabela,como letras, numeros, símbolos e espaços, possui um código binário único associado a ele.
